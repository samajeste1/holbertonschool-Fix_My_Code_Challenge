# Fix My Code Challenge

Fix my code is a new type of project, where we'll jump into an existing code base and fix it!

## Tasks

- `challenge/0-fizzbuzz.py` - FizzBuzz in Python
- `challenge/1-print_square.js` - Print square in JavaScript
- `challenge/2-sort.rb` - Sort in Ruby
- `challenge/3-user.py` - User class in Python
- `challenge/4-delete_dnodeint/` - Double linked list in C
